
Build Instructions:
Linux:
* install clang 3.5+ (with libc++)
* install the opengl and opencl headers and libraries (you need a gl3.h with support for OpenGL 3.2+ and OpenCL headers with support for OpenCL 1.1+)
* install zlib, libpng, libxml2, freetype2
* clone the SDL2 and SDL2_image repositories: hg clone http://hg.libsdl.org/SDL && hg clone http://hg.libsdl.org/SDL_image
* build both of them and install them to /usr
* install premake4 (if your distribution has no package for this, download it manually and copy premake4 to /usr/bin - http://industriousone.com/premake/download)
* depending on whether you're using gcc or clang do a "./premake.sh gcc" or "./premake.sh clang" (if you did all prior steps correctly, this shouldn't print any error)
* "make" (again, this shouldn't print any errors if you followed all previous steps correctly)
* run "./install.sh" (this will install a2elight to /usr/local/) or copy/link all files manually (*.so to /usr/local/lib and a2elight/src to /usr/local/include/a2elight)

OS X:
* install Xcode 6.2+
* install XQuartz http://xquartz.macosforge.org (needed for zlib, libxml2, freetype2)
* clone the SDL2 and SDL2_image repositories: hg clone http://hg.libsdl.org/SDL && hg clone http://hg.libsdl.org/SDL_image
* build both of them and install them to /Library/Frameworks
* alternatively: copy both frameworks from within an .app that uses a2elight to /Library/Frameworks
* open src/a2elight.xcodeproj and build it
* in Terminal do (but change /path/to/a2elight!):
** sudo ln -sf /path/to/a2elight/src /usr/local/include/a2elight
** sudo ln -sf /path/to/a2elight/lib/liba2elight.dylib /usr/local/lib/liba2elight.dylib
** sudo ln -sf /path/to/a2elight/lib/liba2elightd.dylib /usr/local/lib/liba2elightd.dylib

on iOS:
* install Xcode 6.2+
* install floor: https://github.com/a2flo/floor
* compile a2elight.xcodeproj (a2elight_ios target)
* install/symlink a2elight .a and lib folder:
** sudo ln -sf /path/to/a2elight/bin/liba2elight{,d}.a {iPhoneOS/iPhoneSimulator folders}/usr/lib/
** sudo ln -sf /path/to/a2elight {iPhoneOS/iPhoneSimulator folders}/usr/include/a2elight
* linker flags for a depending project: -lSDL2 -lSDL2_image -lfloor -la2elight -lcrypto -lssl

Windows:
* install an x64 MinGW-w64 distribution with gcc 4.9 and full std::thread support (http://mingw-w64.sourceforge.net), and clang 3.5+/libc++ (must be compiled manually)
* install MSYS (http://sourceforge.net/apps/trac/mingw-w64/wiki/MSYS) to /path/to/mingw/msys
* build and install zlib, libpng, libxml2, freetype2, SDL2, SDL2_image
* install the opengl 3.2+ and opencl 1.1+ headers and libraries (renaming might be necessary depending on your distribution and version)
* install premake4 (http://industriousone.com/premake/download)
* set the MINGW64_ROOT environment variable to your specific mingw64 root folder (default is "/c/mingw64/")
* run "./premake.sh gcc" and hope for the best
* binaries (the dll) will be copied/installed to $MINGW64_ROOT/export and all include files will be copied to $MINGW64_ROOT/msys/include
